# Portfolio — Mynul Islam

Single-page responsive portfolio (mobile, tablet, desktop). Static HTML, CSS, and JS.

## What’s included

- **Hero:** Your photo (left on desktop), short intro, LinkedIn & GitHub links  
- **About:** Short “what I do”  
- **Skills:** Tech stack and focus areas  
- **Projects:** List + space for project links  
- **Resume:** Download link for your PDF  
- **Contact:** Email and WhatsApp  

## Before deploying

1. Add **profile.jpg** (your photo) in this folder.  
2. Add **resume.pdf** (your CV) in this folder.  
3. In **index.html**, replace:
   - `YOUR_EMAIL@example.com`
   - `YOUR_WHATSAPP_NUMBER` (e.g. `491234567890`)
   - `YOUR_LINKEDIN_USERNAME`
   - `YOUR_GITHUB_USERNAME`
